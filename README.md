# PAIED Program Website

Official website for the AISOD PAIED (Practical AI Engineering & Development) Program - A free 9-month practical program transforming beginners into AI Engineers/Developers.

## About

The PAIED Program is a comprehensive, hands-on training program designed to take absolute beginners from zero experience to shipping professional-grade AI-powered solutions. The program focuses on practical, project-based learning where success is measured by completed, publicly published projects.

## Features

- Modern Next.js 14 website with TypeScript
- Responsive design with Tailwind CSS
- Markdown-based curriculum content
- PDF download functionality
- Multi-language support
- SEO optimized
- Accessible UI/UX

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Content**: Markdown with React Markdown
- **Deployment**: Netlify (configured)

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aisod/paied.git
cd paied
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
- `npm run debug-build` - Debug build process
